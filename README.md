9. Vercel Deployment

## How To Use :closed_lock_with_key:

```bash
# Clone this repository
$ git clone git@github.com:mux-mux/findface.git

# Go into the repository
$ cd findface

# Install dependencies
$ npm install

# Open browser & listen to js changes
$ npm start
```
